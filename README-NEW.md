## 🚀 Como Executar

1. **Clone o repositório:**
```bash
git clone https://github.com/Alttabcorp/WEB-esports-alttab.git
cd WEB-esports-alttab
```

2. **Inicie um servidor local:**
```bash
# Python 3
python3 -m http.server 8080

# Node.js
npx serve

# PHP
php -S localhost:8080
```

3. **Acesse no navegador:**
```
http://localhost:8080
```
