League of Legends Builder (Versão Modular)

Esta é a nova versão modular do builder de League of Legends, com cada função em arquivos separados para melhor manutenção.

A estrutura do projeto foi reorganizada da seguinte forma:

  • /modules/api/ - Funções de API para acessar Data Dragon
  • /modules/builder/ - Funções para gerenciar campeões, itens e slots
  • /modules/stats/ - Funções para cálculos e exibição de estatísticas
  • /modules/modals/ - Modais para seleção de campeões e itens
  • /modules/utils/ - Funções utilitárias e eventos
  • /modules/config.js - Configurações e constantes globais

Para começar a usar a nova versão, basta importar esta página ou integrar o novo arquivo lol-data-new.js no seu projeto.
